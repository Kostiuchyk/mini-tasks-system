# mini-tasks-system

## Running the Application

### 1. Backend

Requires [Docker](https://www.docker.com/) to be installed.

In the `backend` directory, run:

```bash
docker compose up --build
```

### 2. Frontend

Requires [Node.js](https://nodejs.org/) to be installed.

In the `frontend` directory, run:

```bash
npm install
npm run dev
```

> **Note:** Be sure to open the frontend at http://localhost:5173/
